4🧠 Which model should I use? Practical guidance

Most people don't need to pick a model — Copilot's default routing handles it. But if you want to override, this guide shows which model suits which task. Start with the default. Only switch when there's a clear reason.

⚙️ Auto Default
Copilot reads your prompt and routes it to the right model — fast for simple asks, deeper for complex ones. Leave this on unless you have a reason to override. Covers 80%+ of everyday work.
⚡ Quick Response
Forces Copilot to use a fast, lightweight model (GPT-5 Instant family). Best for quick lookups, short rewrites, and back-and-forth chat where you want speed over depth. Cheapest tier.
🧠 Think Deeper
Forces a reasoning model (GPT-5 Reasoning / Claude Opus). Use for complex problems, multi-step analysis, careful writing, or anything where the answer matters. Slower and uses more compute — pick deliberately.

🟢 GPT-5 family (OpenAI)
The general-purpose default for Copilot Chat. Fast, balanced, well-rounded — Auto-routing covers everyday tasks like summarisation, Q&A, web lookups and light writing. Still a strong override in Excel Agent Mode if Claude isn't producing the result you want. GPT-5 Instant is the cheapest tier; GPT-5 Reasoning is "Think Deeper".
🟣 Claude family (Anthropic)
Now the default reasoning model for Copilot in Excel and PowerPoint, with Word expected to follow in summer 2026. Strong long-form writing, careful reasoning, and a noticeable edge on structured artefacts like HTML, CSS, JSON, YAML and SharePoint formatting. Sonnet 4.6 is the sweet spot for everyday work; Opus 4.8 for deep reasoning and complex structured output (costs more credits). Requires admin opt-in; off by default in UK/EU.
⚪ Microsoft fine-tuned
Cheaper, surface-specific defaults. Cowork 1 (coming soon) for routine Cowork tasks. Use these as the default in their product unless you have a reason to override.
💡 Why does this matter? Microsoft has shifted Office Agent experiences (Excel and PowerPoint, with Word expected to follow in summer 2026) to use Anthropic Claude as the default reasoning model from May 2026, with OpenAI GPT-5 remaining selectable. The standalone Word/Excel/PowerPoint Agents in the M365 Copilot app are Claude-only. For structured artefacts (HTML, CSS, JSON, SharePoint formatting) Claude — especially Opus — has a clear edge. For everyday Copilot Chat tasks outside Office apps, GPT-5 with Auto routing remains the default and is plenty.
